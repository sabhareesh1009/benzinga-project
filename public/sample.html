Benzinga Earnings Widget - Web Component Demo

This page demonstrates how to use the Benzinga Earnings Widget as a Web Component. The widget displays upcoming company earnings organized by day of the week.

Basic Usage

Include the web component script and use the custom element in your HTML:

<script src="/earnings-widget.js"></script>
<benzinga-earnings-widget></benzinga-earnings-widget>

Customization Options

You can customize the widget by setting attributes on the custom element:

Attribute Description Default Example
api-key Your Benzinga API key (required if not using default) - f090a778d74f4450a11ad417ad72740c
title Custom widget title Earnings Calendar Upcoming Earnings
logo Custom logo text BZ ERNS
days-to-show Number of days to display 5 3

Example with Custom Attributes:

<benzinga-earnings-widget title="Upcoming Earnings" logo="ERNS" days-to-show="3"> </benzinga-earnings-widget>

Integration Tips

  • Loading the Script: Include the script in the head or at the end of the body.
  • Styling: The component uses Shadow DOM for style isolation, but you can size the container as needed.
  • API Key: For production use, always provide your own Benzinga API key.
  • Responsive Design: The widget is responsive and will adapt to its container size.
